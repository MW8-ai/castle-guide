# Prompt Pack

First-class product surface: copy-paste prompts so owners can use any LLM without giving Castle Guide an API key.

| Folder | Use |
|--------|-----|
| `ingestion/` | Photos/notes → valid import JSON |
| `enrichment/` | Model lookup, manuals, filters |
| `ideation/` | Room reimagine for image models |
| `builder/` | Extend the app following its own laws |

Manual mode is the zero-cost default forever. BYO-key adapters (Phase 4) must target the same schemas.
